# Dev-Container-Template

**Purpose**

This repository is intended to hold various dev-container configuration files; used for various current and future projects.

**Useage**

Create/find a devcontainer with the correct dependencies/tooling required for the project, save it locally, then all it to the *`.devcontainer/`* folder in your root directory.

**Note**

devcontainer configurations will be built from Dockerfiles, also supplied in the same directory as the corresponding devcontainer.json.
